### 1. ⚠️ PostgreSQL数据库服务未运行
**问题**: 后端无法连接到PostgreSQL数据库
**错误信息**:
```
psycopg2.OperationalError: connection to server at "localhost" (::1), port 5432 failed
```
**解决方案**: 需要启动Docker中的PostgreSQL容器

**操作步骤**:
```bash
# 1. 检查Docker容器状态
docker ps -a

# 2. 启动PostgreSQL容器（如果已存在但未运行）
docker start <container_name>

# 3. 如果容器不存在，重新创建
docker run --name postgres-chat -e POSTGRES_PASSWORD=postgres -e POSTGRES_DB=chat_db -p 5432:5432 -d postgres
```
